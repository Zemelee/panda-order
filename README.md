第七组的项目——“熊猫川菜馆”，咱就是说第七组啥时候儿收购美团啊！！！
家人们，冲啊！！！
目标：
    为用户提供方便快捷的在线点餐服务，并为商家提供高效的菜品管理和销售数据统计功能
用户功能：
    用户注册与登录、菜品浏览与搜索、菜品点餐与购物车、订单查看与支付、会员制度与信息自定义、余额充值、用户反馈、菜品推荐
商家功能：
    订单管理、菜品管理、管理登录日志以及数据统计功能

----------------------------------------------------------------------------------------------------------------------
后端：springboot、mysql、mybatis、阿里云短信服务
前端：vue、ElementUI、Echarts、微信小程序原生、md5

数据库也在里面，把后台和数据库相关的用户名、密码、版本啥的改成与自己一致的应该就可以运行了
